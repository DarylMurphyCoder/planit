# Screenshots Folder

Place your application screenshots here for the main README.md.

## Required Screenshots

1. **login-page.png** - Authentication page with gradient background
2. **task-list.png** - Main dashboard with tasks, filters, and statistics
3. **task-form.png** - Create/edit task form with all fields

## Tips for Taking Screenshots

- Use a clean browser window (no extensions visible)
- Clear resolution (1920x1080 recommended)
- Show real data (sample tasks, not empty state)
- Crop to remove unnecessary browser chrome
- Use PNG format for best quality

## Image Naming Convention

Use lowercase with hyphens:
- ✅ `task-list.png`
- ✅ `login-page.png`
- ❌ `TaskList.PNG`
- ❌ `task_list.jpg`
